# goGPS_Java 项目说明

## 1. 项目概况

goGPS_Java 是基于开源项目 goGPS 的 Java 重写版本，核心功能包括：

- **多星座 GNSS 定位**：支持 GPS、GLONASS、Galileo、BeiDou、QZSS、IRNSS
- **多定位模式**：单点定位（SPP）、伪距双差、载波相位 RTK 卡尔曼滤波
- **多数据格式**：RTCM3、RINEX v2/v3、UBX、NVS、SkyTraq
- **位移解算引擎**：面向形变监测的成套解算→清洗→诊断流水线
- **数据持久化**：设备状态快照、异常事件记录、数据修正记录

## 3. 主要算法

### 3.1 卡尔曼滤波 RTK 定位

**核心类**：[KF_DD_code_phase.java](file:///D:/code/goGPS_Java/src/main/java/org/gogpsproject/positioning/KF_DD_code_phase.java)

双差载波相位+伪距卡尔曼滤波，实现了 RTKLIB 的核心算法：

- **状态向量**：位置(x,y,z) + 速度(vx,vy,vz) + 加速度(ax,ay,az) + 双差模糊度 + 对流层延迟 + 电离层延迟
- **动态模型**：静态(STATIC)、匀速(CONST_SPEED)、匀加速(CONST_ACCELERATION)
- **观测模型**：双差伪距 + 双差载波相位
- **整周模糊度固定**：LAMBDA 方法 + Ratio 检验
- **周跳检测**：基于伪距近似或 Doppler 预报相位
- **电离层/对流层**：支持 IFLC 消电离层组合、实时估计 ZTD
- **不一致性检测**：卡方检验（Chi-Square Innovation Test）
- **发散检测与重置**：连续发散超过阈值自动重置滤波器

### 3.2 单点定位

**核心类**：[LS_SA_code.java](file:///D:/code/goGPS_Java/src/main/java/org/gogpsproject/positioning/LS_SA_code.java)

最小二乘伪距单点定位，支持多种观测值组合：

- 伪距单点定位（CODE_STANDALONE）
- 快照单点定位（CODE_STANDALONE_SNAPSHOT）
- 粗时单点定位（CODE_STANDALONE_COARSETIME）
- 卡尔曼单点滤波（KALMAN_FILTER_CODE_PHASE_STANDALONE）

### 3.3 双差伪距定位

**核心类**：[LS_DD_code.java](file:///D:/code/goGPS_Java/src/main/java/org/gogpsproject/positioning/LS_DD_code.java)

最小二乘双差伪距定位，可作为 RTK 的初始解或独立使用。

### 3.4 五层清洗流水线

**核心类**：[DisplacementCleaner.java](file:///D:/code/goGPS_Java/src/main/java/org/gogpsproject/displacement/cleaning/DisplacementCleaner.java)

| 层级 | 名称 | 算法 | 说明 |
|---|---|---|---|
| 1 | 质量门禁 | 阈值过滤 | 卫星数、PDOP、RMS、Ratio 基本质量检查 |
| 2 | 跳变检测 | 相邻历元差分 | 检测相邻历元位移突变，支持同向趋势快速通道 |
| 3 | 统计粗差 | IQR / 3σ | 滑动窗口内统计粗差检测 |
| 4 | 值替换 | 中位数/均值替换 | 粗差值替换为窗口统计量 |
| 5 | 基线记忆 | 快慢基线对比 | 5分钟快基线 vs 24小时慢基线，检测阶跃 |

### 3.5 设备诊断

**核心类**：[DeviceDiagnostician.java](file:///D:/code/goGPS_Java/src/main/java/org/gogpsproject/displacement/diagnosis/DeviceDiagnostician.java)

- **温漂分析**：温度与位移相关性分析，识别热膨胀效应
- **周期波动检测**：FFT/自相关检测周期性波动模式
- **抖动检测**：高频低幅抖动识别
- **自适应阈值**：诊断结果反馈到清洗阈值，动态调整

### 3.6 数据持久化

**核心类**：[PersistenceCallback.java](file:///D:/code/goGPS_Java/src/main/java/org/gogpsproject/displacement/persistence/PersistenceCallback.java)

三层数据架构（内存缓存 → 持久化存储 → 业务库），通过回调接口解耦：

- 设备状态快照保存/恢复（LRU 淘汰时自动持久化）
- 异常事件记录
- 数据修正记录（支持事后修正）
- 位移结果存档

## 4. 支持的 GNSS 系统

| 系统 | 卫星 PRN | 频率 | 消息类型 |
|---|---|---|---|
| GPS | G01-G32 | L1/L2 | RTCM 1004/1012/1019, RINEX G |
| GLONASS | R01-R24 | L1/L2 | RTCM 1012/1020, RINEX R |
| Galileo | E01-E36 | E1/E5a | RTCM 1046/1045, RINEX E |
| BeiDou | C01-C37 | B1/B2 | RTCM 1042/1043/1044, RINEX C |
| QZSS | J01-J07 | L1/L2 | RTCM 1046, RINEX J |
| IRNSS | I01-I07 | L5 | RTCM 1041, RINEX I |

## 5. 支持的输入格式

| 格式 | 读取器 | 特点 |
|---|---|---|
| RTCM3 | RTCM3FileReader / RTCM3Client | 文件/NTRIP 实时流 |
| RINEX v2/v3 | RinexObservationParser / RinexNavigationParser | 标准观测+星历格式 |
| u-blox UBX | UBXFileReader / UBXSerialConnection | 串口/文件 |
| NVS | NVSFileReader2 / NVSSerialConnection | 串口/文件 |
| SkyTraq | STQFileReader | 文件 |
| SP3 | SP3Parser | 精密星历 |
